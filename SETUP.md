# Mental Health Tracker - Development Setup

## Quick Start

1. **Run the demo version (no setup required)**:
   ```bash
   python3 demo.py
   ```

2. **For full web application (requires Flask)**:
   ```bash
   pip install flask flask-sqlalchemy flask-login
   python3 app.py
   ```
